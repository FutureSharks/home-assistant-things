# home-assistant-things

A collection of things I use in Home Assistant

## Blueprints

1. [dynamic-light](dynamic-light): Automatically controls a dimmable light based on presence, lux and a remote
2. [ikea-bilresa-varmblixt](ikea-bilresa-varmblixt): For the matter over thread devices from Ikea: Bilresa remote and Varmblixt light
3. [rgb-air-quality-monitor](rgb-air-quality-monitor): Periodically reads CO₂, VOC, and NOx sensors, mapping values to an RGB light
